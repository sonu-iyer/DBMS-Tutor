Database Fundamentals

  • Database:

    • A collection of logically related data. Example: A library database storing books, authors, and borrower information.

  • Types of Databses:
    • Relational Database: Uses tables (RDBMS).
    • NoSQL Database: Key-value, document, or graph storage.
    • Graph Database: Handles complex relationships (e.g., social networks).

    Key Terms:

    • Table: Structure to store data in rows and columns.
    • Record: A single row in a table.
    • Field: A single column in a table
